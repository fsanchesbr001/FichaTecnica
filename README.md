# Projeto Ficha Técnica
## Descrição
Este projeto tem como objetivo a criação de uma ficha técnica de um produto, onde é possível adicionar, editar e excluir produtos, além de visualizar a lista de produtos cadastrados.
## Tecnologias
- Java 17
- Spring Boot 3.4.1

## Dependências
- Spring Data JPA
